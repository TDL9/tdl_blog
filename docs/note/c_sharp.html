C#中 #region指令

#region是一个分块预处理命令，它主要用于编辑代码的分段，在编译时会被自动删除。

#region使你可以在使用VS代码编辑器的大纲显示功能时指定可展开或折叠的代码块。

例如： 

#region  REGION_NAME
public class MyClass
{
    static void Main() 
    {
        ...
    }
}
#endregion

备注：#region块必须以#endregion指令终止。

#region块不能与#if块重叠。但是，可以将#region块嵌套在#if块内，或将#if块嵌套在#region块内。
